# Binary to decimal converter
def binary_to_decimal(binary):
    decimal = 0
    power = len(binary) - 1
    for digit in binary:
        decimal += int(digit) * 2**power
        power -= 1
    return decimal

# Decimal to binary converter
def decimal_to_binary(decimal):
    binary = ""
    while decimal > 0:
        remainder = decimal % 2
        binary = str(remainder) + binary
        decimal = decimal // 2
    return binary

  1. What is the difference between boolean operations and logic gates?

Boolean operations are mathematical operations that involve Boolean values (True or False) and are used to determine the truth value of logical expressions. Logic gates, on the other hand, are physical or electronic devices that implement Boolean operations by taking one or more input signals and producing an output signal based on a specific logical function. In short, Boolean operations are abstract concepts used in logic and mathematics, while logic gates are the physical components that implement these concepts in digital circuits.
